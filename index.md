## 2. Термины

Для целей настоящего договора используются следующие основные термины:

- `Сервис` — программный сервис `Реплайнер` (`Repliner`), включая Telegram-бота, иные пользовательские интерфейсы и интеграции с внешними системами, необходимые для его работы.
- `Telegram-бот` — основной пользовательский интерфейс сервиса в мессенджере Telegram, через который Заказчик проходит идентификацию, получает уведомления и использует часть функциональности сервиса.
- `Учетная запись` — совокупность данных Заказчика в сервисе, позволяющая идентифицировать Заказчика и использовать функциональность сервиса.
- `Маркетплейс` — внешняя площадка, интегрированная с сервисом, включая Wildberries, Яндекс Маркет и иные поддерживаемые площадки.
- `Магазин` — аккаунт, кабинет или иная учетная запись продавца на маркетплейсе, подключенная Заказчиком к сервису.
- `Отзыв` — сообщение покупателя на маркетплейсе, полученное сервисом через интеграцию или иным допустимым способом.
- `Ответ` — текст, сформированный сервисом для публикации в ответ на отзыв, включая проект ответа и автоматически опубликованный ответ.
- `Автоответ` — функция автоматической подготовки и/или автоматической отправки ответа на отзыв на основании настроек Заказчика.
- `Подписка` — предоставление доступа к функциональности сервиса на условиях выбранного тарифа в течение соответствующего периода.
- `Период подписки` — временной интервал доступа к сервису на условиях выбранного тарифа, бесплатного, пробного или платного режима.
- `Текущий период` — период подписки, действующий для Заказчика в данный момент времени.
- `Следующий период` — следующий по времени период подписки, который, если это предусмотрено интерфейсом сервиса, может быть заранее настроен Заказчиком без оплаты либо оплачен заранее.
- `Тариф` — набор условий доступа, срок действия, стоимость, лимиты, доступные маркетплейсы, количество мест для магазинов и иные параметры использования сервиса.
- `Место для магазина` — право Заказчика подключить и использовать в рамках действующего периода подписки один Магазин. В интерфейсах сервиса такое место также может именоваться слотом.
- `Расширение` — дополнительная платная опция сервиса, приобретаемая отдельно или вместе с периодом подписки, в том числе для текущего периода, следующего периода либо для конкретного магазина, если это предусмотрено интерфейсом сервиса.
- `Платеж в обработке` — состояние заказа периода подписки или Расширения, по которому Заказчик инициировал оплату, но Исполнитель еще не получил подтверждение успешного платежа от Платежного провайдера.
- `Автопродление` — механизм автоматического оформления и оплаты следующего периода подписки с использованием ранее сохраненного способа оплаты при наличии отдельного волеизъявления Заказчика.
- `Платежный провайдер` — лицо, обеспечивающее прием платежей и, если это предусмотрено интерфейсом сервиса, сохранение способа оплаты для автопродления.
